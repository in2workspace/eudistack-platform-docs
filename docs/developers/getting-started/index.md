Esta guía te lleva de cero a tener tu primera integración funcional con EUDIStack.

## Camino recomendado

1. **[Sandbox](sandbox.md)** — solicita acceso al entorno de pruebas y obtén tus credenciales de desarrollo.
2. **[Tu primera integración](first-integration.md)** — emite y verifica una credencial end-to-end usando los endpoints públicos.
3. **[Conceptos clave](../concepts/index.md)** — entiende los protocolos antes de pasar a producción.
4. **[Guías por caso de uso](../guides/index.md)** — escoge la integración que necesitas (OIDC, SCIM, API).

## Requisitos

- Familiaridad con OAuth 2.0 y JWT.
- Capacidad de exponer un endpoint HTTPS para callbacks (en desarrollo basta con un túnel tipo ngrok).
- Cliente HTTP de tu elección (curl, Postman, código nativo).
