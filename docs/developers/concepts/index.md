# Conceptos

Esta sección cubre los protocolos y patrones que sustentan EUDIStack. No es un curso de identidad digital — es la mínima base que necesitas para integrar con propiedad.

<div class="grid cards" markdown>

-   :material-cloud-upload: [**OID4VCI**](oid4vci.md)

    OpenID for Verifiable Credential Issuance. Cómo se emite una credencial al wallet.

-   :material-cloud-download: [**OID4VP**](oid4vp.md)

    OpenID for Verifiable Presentations. Cómo el wallet presenta credenciales a un verifier.

-   :material-file-key: [**SD-JWT VC**](sd-jwt-vc.md)

    Selective Disclosure JWT — el formato de credencial que usamos por defecto.

-   :material-key-variant: [**DPoP & PKCE**](dpop-pkce.md)

    Cómo se prueba la posesión del token y se protege el intercambio de códigos.

-   :material-domain: [**Multi-tenant**](multi-tenant.md)

    Cómo se aísla cada cliente en EUDIStack y qué implica para tu integración.
