# Estándares

EUDIStack implementa especificaciones normativas. Esta sección las lista con un enlace a la fuente oficial — son consulta on-demand, no las necesitas leer todas para integrar.

## OpenID Foundation

| Especificación | Versión | Descripción |
|---|---|---|
| [OID4VCI](https://openid.net/specs/openid-4-verifiable-credential-issuance-1_0.html) | 1.0 | Emisión de credenciales verificables. |
| [OID4VP](https://openid.net/specs/openid-4-verifiable-presentations-1_0.html) | 1.0 | Presentación de credenciales verificables. |
| [HAIP](https://openid.net/specs/openid4vc-high-assurance-interoperability-profile-1_0.html) | 1.0 | High Assurance Interoperability Profile (perfil eIDAS). |
